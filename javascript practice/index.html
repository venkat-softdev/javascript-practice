Intoduction to javascript

  • javascript is a high-level interpreted programming language.
  • it allows you to add interactivity and dynamic behavior to web pages.
  • javascript is primarily used for client-side scripting in web browsers.
  • it can also be used for server side development with frame works like node.js.
  • javascript was degined to be lightweight and easy to learn.
  • it is one of the most popular programming language in the world.
